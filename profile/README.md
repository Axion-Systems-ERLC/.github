## 🚀 Overview

Axion is a powerful all-in-one Discord bot built for **Roblox and ERLC communities**.

It is designed to replace multiple bots by combining everything into one fast, reliable, and fully customisable system.

From moderation to verification to utilities — Axion handles it all.

---

## ⚙️ What Axion Offers

### 🛡️ Moderation System

Everything you need to manage your server:

* Automod protection
* Warning system
* Kick / ban / timeout tools
* Full logging system
* Role & permission utilities

---

### 🎫 Ticket System

A clean and structured support system:

* Custom ticket panels
* Department routing
* Transcript logging
* Easy staff handling

---

### 🎵 Music System

High-quality music inside Discord:

* Smooth playback
* Queue system
* Simple controls
* Stable performance

---

### 🔐 Roblox Verification

Connect Discord users to Roblox accounts:

* Secure verification system
* Automatic role assignment
* Identity linking for communities

---

### 🚓 ERLC Integration

Built specifically for **Emergency Response: Liberty County** servers:

* Roleplay tools
* Server utilities
* Community management support

---

### ⚡ Utility Features

Extra tools to run your server efficiently:

* Server stats
* Announcements system
* Basic automation tools
* Misc utility commands
